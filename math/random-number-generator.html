Random Number Generator

Generate one or more random numbers within a min/max range.

Result
—

Formula

Random integer = floor(Math.random() × (max − min + 1)) + min

Example

Range 1–100, generate 5 unique numbers: e.g. 7, 42, 18, 91, 3.

How To Use

Set a minimum, maximum, count, and whether repeats are allowed, then click Generate.

Frequently Asked Questions

Is this true randomness? ▾
It uses JavaScript's Math.random(), a pseudo-random generator suitable for general use, not cryptography.
What happens if unique count exceeds the range size? ▾
The calculator will show an error since it can't generate more unique numbers than the range allows.
