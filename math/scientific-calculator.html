Scientific Calculator

A full scientific calculator with trigonometric, logarithmic, power and memory functions — works exactly like a real handheld calculator, entirely offline in your browser.

 
0

How To Use

Tap number and operator keys to build an expression, then press "=" to evaluate. Use SHIFT-style keys sin/cos/tan/log/ln for scientific functions, ( ) for grouping, M+/M-/MR/MC for memory, and AC/DEL to clear. Keyboard input is also supported.

Supported Functions

sin, cos, tan (degrees), log (base 10), ln (natural log), √ (square root), x² , x^y (power), π, e, %, and standard +−×÷ arithmetic with correct operator precedence and parentheses.

Frequently Asked Questions

Does this work offline? ▾
Yes, all calculations run locally in your browser with vanilla JavaScript — no internet or server needed.
Are trig functions in degrees or radians? ▾
Degrees, matching standard handheld scientific calculators.
Can I use my keyboard? ▾
Yes — number keys, + - * / ( ) ., Enter for "=", and Backspace/Escape for delete/clear all work.
