Find the square root (√x) of any number.

Formula

√x = the number that multiplied by itself gives x

Example

√144 = 12. √2 ≈ 1.4142.

How To Use

Enter a non-negative number and click Calculate. Negative numbers return an imaginary result.

Frequently Asked Questions

What is the square root of a negative number? ▾
It's an imaginary number, shown as i multiplied by the square root of the absolute value.
Is the square root always positive? ▾
This calculator shows the principal (positive) square root.
