Basic Calculator

A simple calculator with a real numpad-style keypad — just like a physical PC number pad.

 
0

How To Use

Tap the numpad to build your calculation, then press "=" for the result. C clears everything, ⌫ deletes the last character. Keyboard input (0-9, + - * / . Enter Backspace Escape) also works.

Example

12 + 8 = 20

Frequently Asked Questions

Can I divide by zero? ▾
No, division by zero shows an error instead of a number.
Does it support decimals? ▾
Yes, use the "." key for decimal points.
Can I chain multiple operations? ▾
Yes, e.g. 5 + 3 × 2 = evaluates with standard operator precedence, just like a real calculator.
