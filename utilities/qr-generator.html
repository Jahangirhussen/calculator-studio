QR Code Generator

Generate a scannable QR code from any text or URL — encoded entirely in your browser with a real from-scratch QR algorithm, no external service.

QR Code

How It Works

This tool implements the QR Code encoding algorithm (byte mode, error correction level M) entirely in JavaScript — Reed-Solomon error correction, module placement, and mask pattern selection are all computed locally. No data ever leaves your browser.

Limitations

Supports QR versions 1–10 (up to ~150 alphanumeric or ~90 byte-mode characters depending on content) in byte mode only, which covers the vast majority of URLs and short text. Very long text may not fit.
