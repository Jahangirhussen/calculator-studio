Password Generator

Create strong, random passwords using your browser's cryptographically secure random number generator. Nothing is sent over the network — everything runs locally.

How It Works

This generator uses crypto.getRandomValues(), the Web Crypto API's cryptographically secure pseudorandom number generator (CSPRNG), instead of Math.random(). This means the output is suitable for security-sensitive purposes like passwords, since it cannot be predicted from prior outputs.

Example

With length 16 and all character sets enabled, you might get something like K9#mQ2vL!pR7xT4z. Each character is chosen independently and uniformly at random from your selected character pool.

How To Use

Choose your desired password length with the slider, select which character types to include, then click Generate. Use the Copy Result button to copy the password to your clipboard.

Frequently Asked Questions

Is this password generator secure? ▾
Yes. It uses crypto.getRandomValues(), the same cryptographically secure random source used for keys and tokens, rather than the predictable Math.random() function.
Are my passwords sent anywhere? ▾
No. All generation happens locally in your browser. Nothing is transmitted over the network or stored anywhere.
What length password should I use? ▾
Most security guidance recommends at least 12-16 characters with a mix of character types for strong resistance to brute-force attacks.
