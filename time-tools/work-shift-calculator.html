How It Works

Each Phase is one continuous block of work (24-hour time, e.g. 10:00 to 12:00). Each Break is time subtracted from your worked hours (e.g. lunch, rest). Net Working Time = Total Phase Time − Total Break Time.

Example

Phase 1: 10:00 → 12:00 = 2h. Phase 2: 13:00 → 18:00 = 5h. Total Phase Time = 7h. Break: 13:30 → 14:00 = 30m. Net Working Time = 6h 30m.

Overnight example: Phase 22:00 → 06:00 = 8h (end time earlier than start = counted as next day automatically).

Why 24-hour time?

Using 24-hour time (e.g. 14:00 instead of 2:00 PM) removes the most common input mistake — picking the wrong AM/PM — which used to silently add 12 hours to a result.

Frequently Asked Questions

Why did my old calculation show 14 hours instead of 2? ▾
That happened when using a 12-hour AM/PM selector and picking the wrong AM/PM (e.g. leaving "12:00 AM" instead of "12:00 PM"), which added a full extra 12-hour offset. This version uses 24-hour time only, which removes that entire class of mistake.
Can I add unlimited phases and breaks? ▾
Yes — use "+ Add Phase" and "+ Add Break" as many times as needed, and delete any row you don't need.
Does it handle overnight shifts? ▾
Yes. If an end time is earlier than its start time, it's treated as ending the next day.
