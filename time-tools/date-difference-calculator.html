Find out exactly how many days, weeks, months and years lie between two dates.

How It Works

Enter a start date and an end date. The calculator finds the absolute difference in days using JavaScript's Date object, then converts that into weeks, and separately computes a precise years/months/days breakdown by walking calendar units.

Example

From 2024-01-01 to 2024-03-15 is 74 days (10 weeks, 4 days), which breaks down to 2 months and 14 days.

How To Use

Pick a start date, pick an end date, click Calculate. The order doesn't matter — the result is always a positive difference.

Frequently Asked Questions

Does it count leap years? ▾
Yes, calculations use real calendar dates so leap years are automatically accounted for.
What if I enter the end date before the start date? ▾
The calculator always returns the absolute (positive) difference regardless of order.
