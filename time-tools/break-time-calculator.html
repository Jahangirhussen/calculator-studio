Formula

Net Working Time = Shift Duration − Total Break Time

Example

Shift: 1:00 PM → 11:00 PM (10h). Break 1: 2:00–2:30 PM (30m). Break 2: 5:30–6:00 PM (30m). Break 3: 8:15–8:45 PM (30m). Total breaks = 1h 30m. Net Working Time = 8h 30m.

Frequently Asked Questions

Can I add unlimited breaks? ▾
Yes, use "+ Add Break" for as many breaks as needed.
Does it handle overnight shifts? ▾
Yes, if shift end time is before start time it's treated as ending the next day.
