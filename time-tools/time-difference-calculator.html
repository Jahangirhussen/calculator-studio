Time Difference Calculator

Find the duration between two times of day, with automatic overnight handling.

Duration
—

How It Works

Enter a start and end time in 24-hour format. If the end time is earlier than the start time, the calculator assumes it falls on the next day and adds 24 hours automatically.

Example

Start 23:00, End 03:00 → the shift crosses midnight, so the duration is 4 hours (4h 0m), not a negative number.

How To Use

Select a start time and an end time, then click Calculate to see the duration in hours and minutes plus decimal hours.

Frequently Asked Questions

What happens if the end time is before the start time? ▾
It is treated as an overnight span — the calculator adds 24 hours to correctly compute the duration.
Can I get the result in decimal hours? ▾
Yes, the decimal hours equivalent is shown below the main result.
