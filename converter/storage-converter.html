Data Storage Converter

Convert between bits, bytes, KB, MB, GB and TB (binary, 1024-based convention).

Result
—

Formula

value_in_base = value × factor(unit → byte)
result = value_in_base ÷ factor(target → byte)
1 KB = 1024 bytes, 1 MB = 1024 KB, and so on.

Example

Convert 1 GB to MB: 1 GB × 1073741824 = 1073741824 bytes. 1073741824 ÷ 1048576 = 1024 MB.

How To Use

Enter a value, choose the unit you're converting from and the unit you want, then click Convert.

Frequently Asked Questions

Does this use 1000 or 1024 for each step? ▾
This converter uses the binary convention: each unit is 1024 times the previous one (1 KB = 1024 bytes), matching how operating systems typically report file sizes.
How many bits are in a byte? ▾
One byte equals 8 bits.
