Formula

C → F = C × 9/5 + 32
C → K = C + 273.15
F → C = (F − 32) × 5/9
F → K = (F − 32) × 5/9 + 273.15
K → C = K − 273.15
K → F = (K − 273.15) × 9/5 + 32

Example

Convert 100°C to Fahrenheit: 100 × 9/5 + 32 = 212°F. Convert 100°C to Kelvin: 100 + 273.15 = 373.15 K.

Frequently Asked Questions

Why doesn't temperature use a simple base-unit table? ▾
Celsius and Fahrenheit have offset zero points, so conversion is affine (multiply and add), not a pure multiplication like other unit types. This tool uses direct formulas for each pair instead.
What is absolute zero in each scale? ▾
Absolute zero is 0 Kelvin, which equals -273.15°C and -459.67°F.
